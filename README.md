## 图像识别

* [MediaPipe](https://github.com/google/mediapipe)Google开源的多媒体机器学习模型应用框架。(Google 手部跟踪 人脸检测 3D)
* [chineseocr_lite](https://github.com/ouyanghuiyu/chineseocr_lite) 超轻量级中文ocr (ocr)
